# 📈 Portfolio Optimizer

A Python-based portfolio optimization tool that helps users analyze risk and return for a custom set of stocks.

## 🚀 Features
- Enter your own stock tickers (e.g. INFY, ICICIBANK)
- Generates optimal portfolios using Monte Carlo simulation
- Displays Efficient Frontier
- Shows Max Sharpe portfolio allocation
- Calculates key risk metrics
- Download results as an Excel report

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas, NumPy
- Plotly
- yfinance
